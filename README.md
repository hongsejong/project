### 🧩 주요 구현 기능

- 🛠 **관리자페이지**: 통계 시각화, 문의/신고 관리, 유저 자동 정지
- 💬 **실시간 채팅**: 음성 인식 & 음성 합성 기능
- 🗓 **일정 관리**: FullCalendar + IndexedDB 기반 오늘의 할 일
- 📎 **파일 첨부**: 멀티 파일 업로드 지원
- 📊 **분석**: chart.js 를 이용한 문의 수 시각화
- 🔀 **대시보드 구성**: 드래그 앤 드롭으로 UI 순서 변경
